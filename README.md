#TaskFlow: Assignment Management System
#Overview
TaskFlow is a simple and efficient assignment management system designed to streamline task tracking and delegation. It allows users to create, assign, and monitor tasks with relevant details such as name, due date, description, status, and involved personnel. The system supports role-based access, ensuring that only authorized users can modify tasks while assigned users can update task statuses.

#Features
Task Creation: Add new assignments with essential details (name, due date, description, status, assigned people).
Role-Based Access:
Task Assigners: Can create, edit, and delete tasks.
Assigned Users: Can only view tasks and update their status.
Task Views:
View all assignments in a complete list.
Detailed view of each assignment.
#User Roles
Task Assigners: Manage tasks, assign them to users, and modify details.
Assigned Users: View assigned tasks and update their status.
Tech Stack 
Programming Language: Python
GUI Framework: Tkinter
Database: SQLite3
